## 🌐 GitHub Upload Steps

### 1. **Initialize Git Repository**
```bash
cd "c:/Users/vedan/OneDrive/Desktop/project hospital"
git init
git add .
git commit -m "Initial commit: Sahayadri Hospital Pune website"
```

### 2. **Create GitHub Repository**
- Go to github.com
- Click "New repository"
- Name: `sahayadri-hospital-pune`
- Description: `Modern healthcare website for Sahayadri Hospital Pune`
- Make it Public
- Don't initialize with README (already exists)

### 3. **Push to GitHub**
```bash
git remote add origin https://github.com/Bhairavsingrajput/sahayadri-hospital-pune.git
git branch -M main
git push -u origin main
```

### 4. **Enable GitHub Pages**
- Go to repository Settings
- Scroll to "Pages" section
- Source: Deploy from a branch
- Branch: main
- Folder: /root
- Click Save
- Your site will be live at: `https://bhairavsingrajput.github.io/sahayadri-hospital-pune/`
